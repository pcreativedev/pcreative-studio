# Bundled fonts (Neo-Tokyo theme)

These fonts are bundled so the Neo-Tokyo boot splash / theme render exactly as
designed, offline. All are SIL Open Font License 1.1 (see `OFL.txt`).

- **Zen Dots** — © Zenzo Matoba / Google Fonts. OFL 1.1.
- **Chakra Petch** (Regular/SemiBold/Bold) — © Cadson Demak. OFL 1.1.

JetBrains Mono is NOT bundled; the app falls back to the system monospace
(DejaVu Sans Mono) for terminals/code.
